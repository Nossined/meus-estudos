# meus-estudos
-------
Bem-vindo(a) ao meu repositório central de aprendizado! Este espaço foi criado para organizar todos os meus códigos, exercícios práticos e pequenos projetos desenvolvidos durante a minha formação como desenvolvedor(a).

Aqui você encontrará a evolução dos meus estudos diários, focados em absorver os conceitos na prática e de forma manual.

dedicado apenas pra estudos progressivos próprios, sem ajuda de nenhuma IA.
-------
Estudos focados no desenvolvimento Full Stack completo, cobrindo desde a base da web até aplicações modernas.
* **Front-end:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+), React e Bootstrap.
* **Back-end & Banco de Dados:** Node.js, Express, MongoDB e APIs RESTful.
(Colt Steele bootcamp)
-------
Práticas voltadas para o ecossistema de IA, Ciência de Dados e automação de processos.
* **Foco:** Lógica de programação aplicada, engenharia de prompts, análise de dados e integrações com serviços de IA.
(Santander Bootcamp: Inteligência Artificial (DIO))
-------
Embora eu utilize e reconheça o poder das Inteligências Artificiais no dia a dia dev, **os códigos contidos neste repositório são construídos manualmente**. Acredito que o verdadeiro aprendizado vem de digitar linha por linha, entender os erros no terminal e quebrar a cabeça para fixar a sintaxe e a lógica de programação, afinal nem em vibe coding deve-se terceirizar todo trabalho, um bom futuro especialista precisa ter uma ideia da semantica do que está fazendo, todos projetos que porventura tenham ajuda de IA serao marcados/ flagged em sua descriçao, em casos de utilizar ambientaçao de desenvolvimento compratilhado com IA como aprecio antigravity, codex, etc...
